# Select multi values from datatable

title: Vb.NET Select multiple column
collapse: open
```vb

Using PageInfo As DataTable = ExecuteDataset()..
...
Dim result = From data In PageInfo.AsEnumerable()
			Where data.Field(Of String)("UsageMask") = ""
		     	Select New With {
				.Name = data.Field(Of String)("UsageMask"),
				.Other = data.Field(Of String)("UsageMask")
				}
				
```

var result = from data in mDataTable.AsEnumerable ()
             where data.Field<string>("Code") == txtCode.Text
             select new 
                    { 
                      Description = data.Field<string> ("Description"), 
                      Code = data.Field<string> ("Code") 
                    };

foreach (var res in result) 
{
    // last value always replace `txtxDescription.Text` ?? 
    txtxDescription.Text = res.Description;
    txtxCode.Text = res.Code;
}
