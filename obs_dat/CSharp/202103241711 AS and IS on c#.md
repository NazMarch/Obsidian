#Csharp 

AS используется для приведения типов. 
В отличии от оператора is, который только проверяет равенство типов,
оператор as делает проверку и преобразовывает тип.
Например:
##### Using as
```dotnet
class Program  
{  
	static void Main(string[] args)  
 	{ 
 		object myClass = new MyClass();  
 		object myClass1 = new string($"something");  
 		MyClass.Foo(myClass);  
 		MyClass.Foo(myClass1);  
	}
}  
  
class MyClass  
{  
 public static void Foo(object obj)       
    {  
 		var myClass = obj as MyClass;  
		Console.WriteLine(myClass==null ?   
    		$"Parameter is not member of Myclass" :  
    		"Parameter has type Myclass");
 	}
}
```
Однако, с помощью is так же можно делать преобразование:


##### using is
```dotnet
class Program  
{  
 	static void Main(string\[\] args)  
 	{ 
		object myClass = new MyClass();  
 		object myClass1 = new string($"something");  
 		MyClass.Foo(myClass);  
 		MyClass.Foo(myClass1);  
 	}
}  
  
class MyClass  
{  
 	private void ShowHello() => Console.WriteLine("Hello");  
  
 	public static void Foo(object obj)       
    {  
 		if (obj is MyClass myClass)  
 		{ 
			Console.WriteLine(myClass.GetType());  
			myClass.ShowHello();  
 		}
	}
}
```
Если есть возможность преобразовать - выполниться преобразование
