#WebApp 

NonController - есть класс в папке Controllers, но мы не хотим, чтобы он мог обрабатывать запрос и использоваться как контроллер.

```ad-info
title: Attributes
collapse: open
```dotnet
[NonController] //не обрабатывать запрос и использоваться как контроллер
public class HomeController : Controller


[NonAction] //публичный метод контроллера не рассматривался как действие
public string Hello()

[ActionName("Welcome")] //переименование
public string Hello()
```
