#Vim


Use Esc to change mode of Vim
Vim has two basic states: `normal mode` and `insert mode`
Go to insert mode press - `I`
Return normal mode press - Esc.

Use `h` to move left in the normal mode
Use `l` to move right in the normal mode
Use `j` to move up
Use `k` to move down
Use `a` from normal mode to continue writing in a symbol
after the cursor

Use `O` to go to new line
Use `s` to delete current char
Double `g` moves to start of the document

Use `shift+I` to continue writing from start
Use `shift+A` to go to the end of the line
Use `shift+H` move to the start of the document
Use `shift+L` move to the end of the document

How to save: `Esc-> : w`
How to exit:
        `Esc-> : wq`
        `Esc-> : x`
		
Go to start of each word use `b`  and set cursor in the start of the word moving left
Go to start of each word use `w` and set cursor in the start of the word moving right
Go right and set cursor in the end of word

To move from end word to start of the word press `b` `w` `e`
Use `number`+`l,w...etc`

To set cursor after 3 word we can use `3w` to pass 3 word and set cursor.

use `3`+`i` and write word `example text` then press `esc` to return to normal mode and press enter


To find characters use `f`. You can combine with number.

To jump on { } or [] use `shift`+ 5 
Go to start of the line press 0. 
Go to end of the line press `shift`+4
Find the next occurrence of the word under cursor press `shift`+*
